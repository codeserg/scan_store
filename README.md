# Scan_Store - хранение копий документов и выгрузка их для налоговой инспекции.

Функции программы:
- Документы хранятся у вас на компьютере или на вашем сетевом хранилище
- Структура каталогов позволяет найти документы даже БЕЗ этой программы
- Работает на Widnows и Linux
- Подгружает список клиентов и договоров из вашей учетной системы
- Создает файл-архив документов по фильтру - для налоговой

Возможные различные структуры каталогов. Вы сами задаете формат. Например:
- Наша организация/2024/Клиент Вектор ООО/Акты
- 2024/Наша организация/Акты/Клиент Вектор/1 квартал
- Наша организация/Акты/2024/Клиент Вектор/1 квартал
